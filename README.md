# 📊 FinTrack — Personal Finance Dashboard

A modern, responsive personal finance tracker built with **vanilla HTML, CSS, and JavaScript**.  
FinTrack helps users track income, expenses, savings, and visualize spending using interactive charts.

---

## 🚀 Live Demo

https://jamesx6464-webfun.github.io/FinTrack/

---

## ✨ Features

- 💰 Track income and expenses
- 📊 Real-time balance calculation
- 📈 Interactive spending breakdown chart (Chart.js)
- 🧾 Add and delete transactions
- 💾 Persistent storage using localStorage
- 🎯 Savings goal progress tracker
- 📱 Fully responsive mobile-first design
- 🌙 Modern dark UI

---

## 🛠️ Built With

- HTML5
- CSS3 (modern dark UI + responsive grid)
- JavaScript (vanilla ES6)
- Chart.js (data visualization)
- localStorage (data persistence)
